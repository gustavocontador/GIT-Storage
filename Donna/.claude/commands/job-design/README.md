# Job Design Squad - Comandos Instalados

> Design de cargos via SLII (maturidade D1-D4), Hackman-Oldham e frameworks de competencias.

## Quick Start

```bash
# Orquestrador — design de cargo completo
/job-design:agents:job-design-chief

# Analise funcional (tarefas, dados, pessoas)
/job-design:agents:sidney-fine

# Competencias e motivacao
/job-design:agents:mcclelland
```

## Agentes Disponiveis

| Comando | Papel | Baseado Em |
|---------|-------|------------|
| `/job-design:agents:job-design-chief` | Orchestrator | Coordena design de cargo completo |
| `/job-design:agents:sidney-fine` | Analise Funcional | Sidney Fine — Functional Job Analysis (FJA) |
| `/job-design:agents:mcclelland` | Competencias | David McClelland — Competency Modeling |

## Localizacao

- **Squad completo:** `Squads/job-design/`
