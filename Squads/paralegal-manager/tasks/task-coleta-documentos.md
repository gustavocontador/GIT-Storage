# Task: Coleta de Documentos

**Owner:** Agent responsável pelo processo
**Type:** Documental
**Duration:** 3-7 dias

## Objetivo
Coletar todos documentos necessários antes protocolar.

## Steps
1. Gerar checklist docs por tipo processo (dinâmico)
2. Enviar checklist cliente (Onvio WhatsApp)
3. Acompanhar recebimento docs
4. Validar docs recebidos (formato, validade)
5. Alertar pendências (< 7 dias, < 3 dias, vencido)

## Output
- Dossiê completo
- Pronto para protocolo
