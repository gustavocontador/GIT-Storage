## Notas para a Squad (não incluir para o usuário)

### Cruzamento Interno vs Externo
| Resposta do gestor | Dado externo para cruzar | Fonte |
|--------------------|--------------------------|-------|
| 1.1 (como acha que veem) | Social listening: sentimento real | social-listener |
| 1.3 (melhor/pior que concorrentes) | Competitive analysis | data-miner |
| 2.2 (emoções desejadas) | Emoções reais em comentários | social-listener |
| 2.3 (coerência) | Análise de comunicação vs entrega | data-miner |
| 2.4-2.6 (personalidade) | Percepção de marca em redes sociais | social-listener |
| 3.1 (pelo que é lembrada) | Menções e associações online | exa + social-listener |
| 4.1-4.2 (gaps e dúvidas) | Direcionam a pesquisa externa | research-director |

### Valor do Cruzamento
O principal entregável do nível Bronze é o **Gap Analysis**:
- Onde a visão do gestor COINCIDE com a realidade → confirma posicionamento
- Onde DIVERGE → revela oportunidades ou riscos não percebidos
- Onde há DÚVIDA do gestor → prioridade para pesquisa externa

### Extração de "Verbatims Internos"
- Tratar as respostas do gestor como "visão interna" (não como verbatim tradicional)
- Usar formato: > "{resposta}" — Visão interna do gestor
- Sempre contrastar com dado externo quando disponível
