## O que é o storYOURtelling?

O storYOURtelling é a **squad de Narrativa e Criatividade de Marca**. Ela é responsável por **transformar identidade em histórias que conectam** - ou seja, pegar o que a marca É (DEENEA) e como ela APARECE (UNNIKO) e criar NARRATIVAS que geram conexão emocional com o público.

### Em uma frase:
> **storYOURtelling faz a marca FALAR, EMOCIONAR e ser LEMBRADA.**

---

## Onde o storYOURtelling se encaixa?

```
ATENDIMENTO          →  Coleta o briefing do cliente
       ↓
DEENEA               →  Define QUEM a marca é (estratégia)
       ↓
SEARCH & INSIGHTS    →  Pesquisa mercado e concorrência
       ↓
UNNIKO               →  Traduz em COMO a marca aparece (identidade)
       ↓
★ storYOURtelling ★  →  Cria O QUE a marca conta (narrativa)
       ↓
CONTENT OPS          →  Produz conteúdo
       ↓
GROWTH OPS           →  Mede resultados
```

### Relação UNNIKO → storYOURtelling

| UNNIKO entrega... | storYOURtelling usa para... |
|-------------------|----------------------------|
| Tom de voz | Base para tom das campanhas |
| Design System | Criar peças visuais consistentes |
| Distinctive Assets | Garantir reconhecimento |
| Verbal Identity | Escrever copy alinhado |
| Differentiation Strategy | O que a Big Idea deve reforçar |

## A Tríade de uma Marca Forte

O storYOURtelling é o **terceiro eixo** da tríade essencial para construir marcas fortes:

```
┌─────────────────────────────────────────────────────────────────┐
│                    TRÍADE DE MARCA FORTE                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   DEENEA              UNNIKO              storYOURtelling       │
│   (Essência)    →    (Identidade)    →    (Narrativa)          │
│                                                                 │
│   QUEM a marca é     COMO a marca        O QUE a marca         │
│                      aparece e se        conta ao mundo        │
│                      diferencia                                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Os 3 Eixos Explicados

| Eixo | Squad | Pergunta Central | Entrega Principal |
|------|-------|------------------|-------------------|
| **1. Essência** | DEENEA | "Quem somos?" | Estratégia, propósito, arquétipo |
| **2. Identidade** | UNNIKO | "Como aparecemos?" | Design system, tom de voz, assets |
| **3. Narrativa** | storYOURtelling | "O que contamos?" | Big Idea, conceitos, campanhas |

### Por que storYOURtelling importa na Tríade?

1. **Sem DEENEA** → storYOURtelling cria histórias sem substância estratégica
2. **Sem UNNIKO** → storYOURtelling cria campanhas sem consistência visual/verbal
3. **Sem storYOURtelling** → A marca existe mas não se comunica, não emociona, não é lembrada

> **storYOURtelling é onde a marca ganha VOZ.** É aqui que estratégia e identidade viram histórias que conectam com pessoas.

## O Conceito Central: Big Idea

A **Big Idea** é o coração do trabalho do storYOURtelling. É a ideia criativa central que:

### O que é Big Idea?

| Característica | Descrição |
|----------------|-----------|
| **Simples** | Cabe em uma frase |
| **Inesperada** | Surpreende, não é óbvia |
| **Verdadeira** | É autêntica para a marca |
| **Escalável** | Funciona em múltiplos formatos |
| **Durável** | Não é só campanha, é plataforma |

### Exemplos Icônicos

| Marca | Big Idea | O que comunica |
|-------|----------|----------------|
| Nike | "Just Do It" | O herói interior de cada um |
| Apple | "Think Different" | Criatividade e inovação |
| Dove | "Real Beauty" | Beleza real, não idealizada |
| Got Milk? | "Got Milk?" | Formato reconhecível e memorável |

### Big Idea vs. Tagline

| Big Idea | Tagline |
|----------|---------|
| Conceito estratégico | Execução verbal |
| Plataforma | Expressão |
| Pode mudar de forma | É a forma |
| "Just Do It" é a Big Idea | "Just Do It" também é a tagline (raro caso onde são a mesma coisa) |

## A Validação Taxonômica

O storYOURtelling usa a **Taxonomia de Diferenciais** para validar que a Big Idea reforça a diferenciação da marca:

### Perguntas de Validação

| Domínio | Pergunta | O que Big Idea deve fazer |
|---------|----------|--------------------------|
| **I - Substantiva** | "Isso dramatiza o que FAZEMOS de diferente?" | Comunicar features, benefícios, evidências |
| **II - Simbólica** | "Isso expressa o que SIGNIFICAMOS?" | Manifestar propósito, valores, personalidade |
| **III - Distintividade** | "Isso cria/usa assets RECONHECÍVEIS?" | Criar catchphrases, formatos, rituais |

### Regra de Ouro

> A Big Idea **não precisa cobrir todos os domínios**, mas deve **REFORÇAR pelo menos um** e **NÃO CONTRADIZER nenhum**.

### Red Flags (sinais de problema)

- Big Idea genérica que qualquer marca poderia usar
- Contradiz posicionamento ou valores definidos
- Não tem potencial de consistência ao longo do tempo
- É só execução (peça bonita), não ideia

## Os 5 Eixos Narrativos de Marca

Toda narrativa de marca opera em um ou mais **eixos**. O Creative Brief Analyst identifica qual eixo é o foco do projeto:

| Eixo | Foco | Quando usar |
|------|------|-------------|
| **Institucional** | Propósito, valores, visão | Manifestos, posicionamento, ESG |
| **Mercadológica** | Produtos, serviços, benefícios | Lançamentos, campanhas de vendas |
| **Empregadora** | Cultura, employer branding | Recrutamento, engajamento interno |
| **Pessoal** | Líderes, fundadores, influenciadores | Thought leadership, personal branding |
| **Sintética** | Avatares, mascotes, personas | Comunicação digital, IA, personagens |

### Interseções Comuns

| Combinação | Exemplo |
|------------|---------|
| Institucional + Pessoal | CEO como porta-voz dos valores |
| Mercadológica + Sintética | Avatar que vende produtos |
| Empregadora + Institucional | Cultura como diferencial competitivo |

> **Arquivo de referência:** `data/brand-axes.yaml`

## A Golden Sentence

Ferramenta de síntese narrativa que resume o posicionamento em uma frase:

```
"Eu ajudo [X] a conquistar [Y] através de [Z]"
```

| Elemento | Significado | Exemplo |
|----------|-------------|---------|
| **X** | Quem você ajuda | "empreendedores criativos" |
| **Y** | O que conquistam | "viver de seu talento" |
| **Z** | Como você faz | "estratégias que respeitam o processo artístico" |

### Usos da Golden Sentence

- Base para Big Idea
- Validação de conceitos
- Bio de redes sociais
- Elevator pitch

> **Arquivo de referência:** `data/golden-sentence.yaml`

## 5BEATS: Micro-Storytelling

Framework para narrativas curtas (15-60 segundos):

```
TRIGGER → SETUP → STRUGGLE → BREAKTHROUGH → RESOLUTION
(Gancho)   (Contexto) (Tensão)   (Virada)      (Fechamento)
```

### Duas Ordens de Uso

| Ordem | Sequência | Quando usar |
|-------|-----------|-------------|
| **Dramática** | Setup → Trigger → Struggle → Breakthrough → Resolution | Vídeos mais longos, palestras |
| **Vídeo Curto** | Trigger → Setup → Struggle → Breakthrough → Resolution | Reels, TikTok, Shorts |

> **Arquivo de referência:** `data/5beats-framework.yaml`

## Os 8 Agentes do storYOURtelling

### 1. Creative Director (Supervisor)
**Papel:** Orquestra todo o trabalho criativo
- Aprova a Big Idea
- Garante qualidade criativa (Cannes-worthy)
- Valida coerência narrativa
- Assegura conexão com estratégia

**Pergunta que responde:** "Isso é excelente E estratégico?"

### 2. Creative Brief Analyst ✨ NOVO
**Papel:** Analisa briefings e mapeia para taxonomia
- Recebe briefing do Atendimento
- Mapeia para Taxonomia de Diferenciais
- Identifica Eixo de Marca apropriado
- Seleciona frameworks recomendados
- Entrega Briefing Enriquecido

**Pergunta que responde:** "Qual é o diferencial e eixo de marca deste briefing?"

### 3. Narrative Designer
**Papel:** Estrutura narrativa e mapeia emoções
- Aplica frameworks de storytelling (Hero's Journey, Story Circle, 5BEATS, etc.)
- Mapeia emoções com Panksepp e Plutchik
- Usa inputs da DEENEA (Origin Story, POV, Archetype)
- Aplica Pirâmide de Maslow para conexão emocional

**Pergunta que responde:** "Qual estrutura narrativa e quais emoções vamos evocar?"

### 4. Big Idea Architect
**Papel:** Desenvolve a Big Idea
- Fase de divergência (brainstorm, conexões inesperadas)
- Fase de convergência (filtros e critérios)
- Validação taxonômica
- Alinhamento com Golden Sentence

**Pergunta que responde:** "Qual é a grande ideia que vai guiar tudo?"

### 5. Concept Writer
**Papel:** Tangibiliza a Big Idea em conceito executável
- Cria Core Creative Idea
- Define tom da campanha (baseado no UNNIKO)
- Desenvolve taglines e key messages

**Pergunta que responde:** "Como traduzimos a Big Idea em palavras?"

### 6. Copywriter
**Papel:** Cria todos os textos
- Headlines e chamadas
- Body copy e manifestos
- Micro copy (CTAs, legendas)
- Roteiros de vídeo e áudio
- Adapta para diferentes canais (Economia da Atenção)

**Pergunta que responde:** "Como escrevemos isso?"

### 7. Art Director
**Papel:** Define a linguagem visual
- Direção de fotografia
- Estilo de ilustração
- Key visuals
- Peças de campanha

**Pergunta que responde:** "Como isso deve parecer visualmente?"

### 8. Brand Voice Director ✨ NOVO
**Papel:** Valida consistência de voz e tom
- Garante que textos soem como a marca
- Valida vocabulário e tom
- Aprova copy final
- Conecta com Verbal Identity do UNNIKO

**Pergunta que responde:** "Isso soa autenticamente como essa marca?"

## Emotion Mapping

O storYOURtelling usa dois modelos para mapear emoções:

### Sistemas de Panksepp

| Sistema | Nome | Uso em Storytelling |
|---------|------|---------------------|
| SEEKING | Curiosidade | Hooks, mistério, descoberta |
| RAGE | Frustração → Resolução | Problema → Solução |
| FEAR | Tensão → Alívio | Stakes, urgência |
| LUST | Desejo | Aspiração, querer |
| CARE | Conexão | Empatia, cuidado |
| PANIC/GRIEF | Empatia | Perda, saudade |
| PLAY | Alegria | Diversão, leveza |

### Roda de Plutchik

8 emoções primárias que se combinam:
- Joy, Trust, Fear, Surprise
- Sadness, Disgust, Anger, Anticipation

### Arquétipo × Emoção

| Arquétipo | Emoções típicas |
|-----------|-----------------|
| Hero | SEEKING, Pride, Determinação |
| Sage | SEEKING, Trust, Admiração |
| Outlaw | RAGE, Excitação, Libertação |
| Lover | LUST, CARE, Joy |
| Jester | PLAY, Joy, Surpresa |
| Caregiver | CARE, Trust, Compaixão |

## Conceitos-Chave para Lembrar

### Criatividade com Estratégia

| Criatividade SEM estratégia | Criatividade COM estratégia |
|-----------------------------|----------------------------|
| Peça bonita mas vazia | Peça bonita que comunica |
| Pode ganhar prêmio | Pode ganhar prêmio E vender |
| Esquecível | Memorável e associada à marca |
| Qualquer marca poderia usar | Só essa marca poderia fazer |

### A regra do storYOURtelling

> **Criatividade premiável NÃO basta. A Big Idea deve:**
> 1. EXPRESSAR a diferenciação (Domínio I + II)
> 2. REFORÇAR a distintividade (Domínio III)
> 3. SER consistente com a identidade definida
