## Legenda

| Símbolo | Significado |
|---------|-------------|
| `┌──┐` | Processo/Stage |
| `◇` | Ponto de decisão (checkpoint) |
| `★` | CBO Gate (validação obrigatória) |
| `──NO──▶` | Fluxo de rejeição/retorno |
| `YES` | Aprovado, segue para próximo |
| `◄──` | Indica cargo de Director |
| `▼` | Fluxo descendente |
