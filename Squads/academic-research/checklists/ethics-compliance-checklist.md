# Ethics Compliance Checklist

> Validar ANTES de iniciar coleta de dados e ANTES de submissão.
> Usado por: research-chief (ethics-approval), peer-reviewer (review)

---

## Pre-Data Collection (MANDATORY)

- [ ] **IRB/CEP approval obtained** — Número de protocolo documentado
- [ ] **Informed consent form** — Aprovado pelo comitê de ética
- [ ] **Consent process defined** — Written, verbal, or online (per design)
- [ ] **Voluntary participation** — Participantes podem desistir sem penalidade
- [ ] **Risk assessment** — Riscos identificados e mitigados
- [ ] **Benefit assessment** — Benefícios potenciais documentados
- [ ] **Vulnerable populations** — Proteções extras se aplicável (menores, pacientes, etc.)

## Data Privacy & Security

- [ ] **Anonymization plan** — Como dados serão desidentificados
- [ ] **Data storage** — Onde e como dados serão armazenados (segurança)
- [ ] **Access control** — Quem tem acesso aos dados
- [ ] **Retention period** — Quanto tempo dados serão mantidos
- [ ] **Destruction plan** — Como dados serão destruídos após o período
- [ ] **GDPR/LGPD compliance** — Se aplicável ao local/participantes
- [ ] **Data sharing plan** — Se dados serão compartilhados (Open Science)

## During Data Collection

- [ ] **Consent obtained** — De cada participante, ANTES de coletar dados
- [ ] **Confidentiality maintained** — Dados nunca ligados a identidades
- [ ] **Debrief provided** — Participantes informados após participação
- [ ] **Adverse events** — Protocolo para lidar com situações adversas
- [ ] **Protocol followed** — Nenhum desvio do protocolo aprovado pelo IRB

## Publication Ethics

- [ ] **Authorship criteria** — ICMJE: contribuição substantiva + draft + aprovação + accountability
- [ ] **No ghost authorship** — Todos que contribuíram estão listados
- [ ] **No gift authorship** — Ninguém listado sem contribuição real
- [ ] **Conflicts of interest** — Todos declarados (financial, personal, institutional)
- [ ] **Funding declared** — Todas as fontes de financiamento listadas
- [ ] **No duplicate submission** — Manuscrito não submetido a outro journal simultaneamente
- [ ] **Prior publication** — Dados/texto não publicados previamente (ou devidamente citados)

## Systematic Review Ethics

- [ ] **Protocol registered** — PROSPERO registration (pre-search)
- [ ] **Deviations documented** — Qualquer mudança do protocolo registrada
- [ ] **No selective reporting** — Todos os resultados reportados (positivos e negativos)
- [ ] **PRISMA compliance** — Checklist completa (27 items)

## Special Considerations

### Clinical Research
- [ ] Clinical trial registered (ClinicalTrials.gov, ISRCTN)
- [ ] CONSORT checklist followed (for RCTs)
- [ ] Independent DSMB if applicable

### Research with Minors
- [ ] Parental/guardian consent obtained
- [ ] Child assent obtained (age-appropriate)
- [ ] Additional safeguarding measures in place

### Research with Patient Data
- [ ] HIPAA compliance (USA) or equivalent
- [ ] Hospital/institution approval
- [ ] De-identification verified by independent party
