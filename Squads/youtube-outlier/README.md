World-class YouTube strategy & channel growth team for YouTube creators.

## Squad Overview

4 specialist agents + 1 strategy chief orchestrator, each powered by proven YouTube growth frameworks from the world's top creators and strategists, all calibrated to the creator's voice DNA.

## Agents

| Agent | Tier | Methodology | Covers |
|-------|------|-------------|--------|
| `strategy-chief` | Orchestrator | Routing + Strategy QC | Routes requests, enforces quality gates, validates strategy |
| `channel-strategist` | T0 - Diagnostic | Paddy Galloway 7 Pillars | Channel audits, positioning, growth roadmaps |
| `algorithm-expert` | T1 - Strategy | YouTube Algorithm + vidIQ | SEO, discoverability, algorithm optimization, policy compliance |
| `content-strategist` | T1 - Strategy | MrBeast + Think Media | Content pillars, video ideas, viral hooks, series planning |
| `analytics-analyst` | T2 - Analysis | Data-driven optimization | Performance analysis, benchmarks, A/B testing, iteration |

## Workflow

```
Request -> strategy-chief routes ->
  1. channel-strategist audits channel state (BLOCKING)
  2. algorithm-expert optimizes discoverability (if needed)
  3. content-strategist plans content pillars and ideas
  4. analytics-analyst tracks performance and iterates
  5. strategy-chief validates strategy alignment (BLOCKING)
-> Validated strategy delivered
```
