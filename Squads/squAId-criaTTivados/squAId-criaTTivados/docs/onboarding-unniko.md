## O que é o UNNIKO?

O UNNIKO é a **squad de Identidade e Diferenciação de Marca**. Ela é responsável por **traduzir a estratégia em identidade tangível** - ou seja, transformar o que a marca É (definido pela DEENEA) em como a marca APARECE e se DIFERENCIA no mundo.

### Em uma frase:
> **UNNIKO faz a marca ser RECONHECIDA, DIFERENCIADA e CONSISTENTE.**

---

## Onde o UNNIKO se encaixa?

```
ATENDIMENTO          →  Coleta o briefing do cliente
       ↓
DEENEA               →  Define QUEM a marca é (estratégia)
       ↓
SEARCH & INSIGHTS    →  Pesquisa mercado e concorrência
       ↓
★ UNNIKO ★           →  Traduz em COMO a marca aparece (identidade)
       ↓
storYOURtelling      →  Cria campanhas e narrativas
       ↓
CONTENT OPS          →  Produz conteúdo
       ↓
GROWTH OPS           →  Mede resultados
```

### Relação DEENEA → UNNIKO

| DEENEA define... | UNNIKO traduz em... |
|------------------|---------------------|
| Propósito | Messaging e manifesto |
| Arquétipo | Tom de voz e estilo visual |
| Valores | Comportamentos e guidelines |
| Personalidade | Design System |

## A Tríade de uma Marca Forte

O UNNIKO é o **segundo eixo** de uma tríade essencial para construir marcas fortes e consistentes:

```
┌─────────────────────────────────────────────────────────────────┐
│                    TRÍADE DE MARCA FORTE                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   DEENEA              UNNIKO              storYOURtelling       │
│   (Essência)    →    (Identidade)    →    (Narrativa)          │
│                                                                 │
│   QUEM a marca é     COMO a marca        O QUE a marca         │
│                      aparece e se        conta ao mundo        │
│                      diferencia                                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Os 3 Eixos Explicados

| Eixo | Squad | Pergunta Central | Entrega Principal |
|------|-------|------------------|-------------------|
| **1. Essência** | DEENEA | "Quem somos?" | Estratégia de marca, propósito, arquétipo, personalidade |
| **2. Identidade** | UNNIKO | "Como aparecemos e nos diferenciamos?" | Design system, tom de voz, assets distintivos |
| **3. Narrativa** | storYOURtelling | "O que contamos ao mundo?" | Big Idea, conceitos criativos, campanhas |

### Fluxo entre os Eixos

```
DEENEA                    UNNIKO                    storYOURtelling
───────                   ──────                    ───────────────

Propósito         →       Manifesto         →       Big Idea
Arquétipo         →       Tom de Voz        →       Conceito Criativo
Personalidade     →       Design System     →       Identidade Visual de Campanha
Valores           →       Guidelines        →       Narrativas e Histórias
```

### Por que a Tríade importa?

1. **Sem DEENEA** → UNNIKO cria identidade sem fundamento estratégico
2. **Sem UNNIKO** → storYOURtelling cria campanhas sem consistência visual/verbal
3. **Sem storYOURtelling** → A identidade existe mas não é comunicada

> **A força de uma marca está na integração dos 3 eixos.** UNNIKO é a ponte que traduz estratégia (DEENEA) em elementos tangíveis que serão usados na comunicação (storYOURtelling).

### Relação UNNIKO → storYOURtelling

| UNNIKO entrega... | storYOURtelling usa para... |
|-------------------|----------------------------|
| Tom de voz | Definir o tom das campanhas |
| Design System | Criar peças visuais consistentes |
| Distinctive Assets | Garantir reconhecimento |
| Verbal Identity | Escrever copy alinhado com a marca |
| Brand Guidelines | Manter consistência em todas as peças |

## A Taxonomia de Diferenciais

O UNNIKO trabalha com um framework chamado **Taxonomia de Diferenciais de Marca**, que organiza tudo em 3 domínios:

### Os 3 Domínios

| Domínio | Pergunta | O que cobre |
|---------|----------|-------------|
| **I - Substantiva** | O que a marca FAZ de diferente? | Produto, modelo, evidências |
| **II - Simbólica** | O que a marca SIGNIFICA? | Propósito, personalidade, valores |
| **III - Distintividade** | Como a marca é RECONHECIDA? | Assets visuais, verbais, comportamentais |

### Estrutura completa

```
DOMÍNIO I - DIFERENCIAÇÃO SUBSTANTIVA
├── Classe 1: Produto/Serviço
├── Classe 2: Modelo de Negócio
└── Classe 3: Evidências

DOMÍNIO II - DIFERENCIAÇÃO SIMBÓLICA
├── Classe 4: Identidade Narrativa
├── Classe 5: Personalidade
└── Classe 6: Cultura e Valores

DOMÍNIO III - DISTINTIVIDADE
├── Classe 7: Assets Visuais
├── Classe 8: Assets Sensoriais
└── Classe 9: Assets Comportamentais
```

## Os 8 Agentes do UNNIKO

### 1. Brand Director (Supervisor)
**Papel:** Orquestra todo o trabalho do UNNIKO
- Garante que todos os domínios estejam cobertos
- Valida qualidade dos outputs
- Aprova entregas finais

### 2. Brand Auditor
**Papel:** Diagnostica a marca atual
- Avalia a marca nas 27 ordens da taxonomia
- Identifica forças e gaps
- Cria relatório de auditoria

**Pergunta que responde:** "Onde a marca está forte e fraca hoje?"

### 3. Competitive Analyst
**Papel:** Analisa a concorrência pela taxonomia
- Mapeia concorrentes nos 3 domínios
- Identifica white spaces (espaços vazios)
- Encontra oportunidades de diferenciação

**Pergunta que responde:** "Onde podemos ser diferentes dos concorrentes?"

### 4. Differentiation Strategist
**Papel:** Define a estratégia de diferenciação
- Escolhe em quais ordens competir
- Define a promessa de marca
- Cria o mapa de diferenciação

**Pergunta que responde:** "Em que vamos ser únicos?"

### 5. Verbal Identity Specialist
**Papel:** Cria a identidade verbal
- Naming e arquitetura de nomes
- Tom de voz e guidelines verbais
- Vocabulário e linguagem da marca

**Pergunta que responde:** "Como a marca FALA?"

### 6. Design System Architect
**Papel:** Cria o sistema visual
- Logo e marca gráfica
- Paleta de cores
- Tipografia
- Estilo visual

**Pergunta que responde:** "Como a marca PARECE?"

### 7. Identity Architect
**Papel:** Cria assets sensoriais e compila o Brand Book
- Sons, texturas, formatos
- Integração de todos os elementos
- Documentação final

**Pergunta que responde:** "Quais elementos tornam a marca RECONHECÍVEL?"

### 8. Brand Experience Designer
**Papel:** Define a experiência de marca
- Rituais e comportamentos
- Touchpoints e jornada
- Estilo de interação

**Pergunta que responde:** "Como é INTERAGIR com a marca?"


## Conceitos-Chave para Lembrar

### Diferenciação vs. Distintividade

| Diferenciação | Distintividade |
|---------------|----------------|
| Ser MELHOR ou DIFERENTE | Ser RECONHECIDO |
| "Por que escolher?" | "Como identificar?" |
| Domínios I e II | Domínio III |
| Cria preferência | Cria lembrança |

### A regra dos domínios

- **Domínio I:** Ter 2-3 ordens com superioridade VERIFICÁVEL
- **Domínio II:** Coerência narrativa - ordens devem REFORÇAR a mesma tese
- **Domínio III:** Assets ÚNICOS, CONSISTENTES no tempo e DEFENSÁVEIS
