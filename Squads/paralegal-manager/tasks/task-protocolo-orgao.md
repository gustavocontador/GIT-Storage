# Task: Protocolo em Órgão

**Owner:** Agent especialista
**Type:** Administrative
**Duration:** 1 dia

## Objetivo
Protocolar processo em órgão competente (Junta, Prefeitura, MAPA, etc.).

## Steps
1. Validar docs completos (checklist 100%)
2. Preencher formulários online/físicos
3. Pagar taxas (DARE, guias)
4. Protocolar (online ou presencial)
5. Obter número protocolo
6. Registrar em sistema (tracking)

## Output
- Protocolo concluído
- Número protocolo registrado
- Prazo estimado análise
