# Pre-Submission Checklist

> Validar ANTES de submeter qualquer manuscrito para publicação.
> Usado por: research-chief (final review), peer-reviewer (internal review)

---

## Manuscript Quality (research-chief + peer-reviewer)

- [ ] **Research question clara** — SMART/PICO framework aplicado
- [ ] **IMRaD structure completa** — Introduction, Methods, Results, Discussion presentes
- [ ] **Abstract estruturado** — Background, Methods, Results, Conclusion (150-300 palavras)
- [ ] **Title informativo** — Conciso, keywords, 10-15 palavras
- [ ] **Keywords selecionadas** — 3-6 keywords relevantes
- [ ] **Limitations discutidas** — Seção dedicada no Discussion
- [ ] **Implications presentes** — Theoretical + practical implications

## Methodology (research-chief)

- [ ] **Design claramente descrito** — Tipo de estudo, abordagem, justificativa
- [ ] **Sample size justificado** — Power analysis (quantitative) ou saturação (qualitative)
- [ ] **Instrumentos validados** — Usar instrumentos publicados quando possível
- [ ] **Procedimentos replicáveis** — Detalhes suficientes para reprodução
- [ ] **Análise apropriada** — Testes estatísticos corretos para tipo de dados
- [ ] **Ethics approval documentada** — Número do IRB/CEP mencionado

## Citations (citation-manager)

- [ ] **Estilo consistente** — Todo o manuscrito segue um único estilo
- [ ] **DOIs incluídos** — Todos os artigos com DOI têm o link
- [ ] **1:1 match** — Toda citação no texto está na lista de referências (e vice-versa)
- [ ] **Fontes recentes** — Maioria das citações dos últimos 5 anos
- [ ] **Sem excesso de auto-citação** — Citações próprias justificadas
- [ ] **Plagiarism check** — Similarity < 15% (excluindo referências)

## Formatting (academic-writer)

- [ ] **Word count dentro do limite** — Verificar guidelines do journal
- [ ] **Tabelas formatadas** — Títulos acima, notas abaixo, sem linhas verticais
- [ ] **Figuras de alta resolução** — Mínimo 300 DPI
- [ ] **Captions completas** — Tabelas e figuras auto-explicativas
- [ ] **Numeração sequencial** — Tabelas, figuras, equações numeradas em ordem
- [ ] **Tense correto** — Past (methods/results), Present (introduction/discussion)

## Journal-Specific (research-chief)

- [ ] **Instructions for Authors** — Lidas e seguidas
- [ ] **Template do journal** — Usado se disponível
- [ ] **Cover letter** — Personalizada para o journal
- [ ] **Author contributions** — CRediT taxonomy ou similar
- [ ] **Conflicts of interest** — Declarados (mesmo que "none")
- [ ] **Data availability** — Statement incluído
- [ ] **Supplementary materials** — Preparados se necessário

## Final Checks (peer-reviewer)

- [ ] **Peer review interno** — Pelo menos 1 revisor antes de submeter
- [ ] **Spelling/grammar** — Verificado (Grammarly ou similar)
- [ ] **Consistency** — Termos, abreviações, formatting consistentes
- [ ] **File format** — .docx ou LaTeX conforme exigido
- [ ] **All co-authors** — Aprovaram a versão final
