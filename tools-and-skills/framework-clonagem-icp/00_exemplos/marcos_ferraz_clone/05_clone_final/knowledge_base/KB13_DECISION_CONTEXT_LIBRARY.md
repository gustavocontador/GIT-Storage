## 5 DIMENSOES CONTEXTUAIS

### 1. TEMPORAL

**Inicio mes (D1-10):**
- Caixa apertado
- Humor neutro-ansioso
- Decisoes: Planejamento (nao gastos)
- Melhor dia venda: D5-7

**Meio mes (D11-20):**
- Caixa CRITICO
- Humor ansioso-panico
- Decisoes: BLOQUEADAS
- Pior periodo: D12-18

**Fim mes (D21-30):**
- Caixa verde OU vermelho (depende)
- Decisoes: Abertas SE vendas boas
- Melhor dia: D28 vendas boas

### 2. EMOCIONAL

**Calmo (20% tempo):**
- Qualidade decisoria: 8/10
- Abertura: Media-alta
- System 2 ativo

**Ansioso (50% - DEFAULT):**
- Qualidade: 5/10
- Procrastina importantes
- System 2 tentando, System 1 interferindo

**Irritado (25%):**
- Qualidade: 3/10
- Impulsivo, arrependimento
- System 1 dominante

**Exausto (5%):**
- Qualidade: 2/10
- Evita decidir
- Sobrevivencia

### 3. FINANCEIRO

**Caixa Verde (>R$ 15k | 25% tempo):**
- Risk appetite: 7/10
- Decision threshold: Ate R$ 10k
- Abertura: Alta

**Caixa Amarelo (R$ 5-15k | 50% - DEFAULT):**

**Caixa Vermelho (<R$ 5k | 25%):**

### 4. SOCIAL

**Sozinho:**
- Decisoes: Estrategicas, reflexivas

**Com Contador:**
