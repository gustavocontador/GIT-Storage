# Quality Shield Squad - Comandos Instalados

> QA automatizado: diagnostico de bugs, analise de impacto, regressao, verificacao em producao.

## Quick Start

```bash
# Orchestrator — triagem e roteamento
/quality-shield:agents:chief

# Diagnosticar bug ou problema
/quality-shield:agents:diagnostician

# Analise de impacto de mudanca
/quality-shield:agents:impact-analyst
```

## Agentes Disponiveis

| Comando | Papel | Especialidade |
|---------|-------|---------------|
| `/quality-shield:agents:chief` | Orchestrator | Triagem, roteamento, decisao |
| `/quality-shield:agents:diagnostician` | Diagnostico | Root cause analysis, deep trace |
| `/quality-shield:agents:impact-analyst` | Impacto | Analise de blast radius, dependencias |
| `/quality-shield:agents:regression-guard` | Regressao | Testes de regressao, guard rails |
| `/quality-shield:agents:production-monitor` | Producao | Health checks, verificacao pos-deploy |

## Localizacao
