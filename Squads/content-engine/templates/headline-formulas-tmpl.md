# Headline Formulas Template

**Pattern:** CP-PP-004
**Source:** Gary Halbert + David Ogilvy + Claude Hopkins

---

## Regras de Ogilvy para Headlines

1. 5x mais pessoas leem headlines que body copy
2. Inclua beneficio na headline
3. Use palavras de novidade ("novo", "agora", "finalmente")
4. Seja especifico com numeros
5. Headlines longas vendem mais que curtas (6-12 palavras = sweet spot)
6. Gaste 50% do tempo na headline

---

## 20 Formulas Comprovadas

### Resultado Direto

```
1. "Como [resultado] em [prazo] sem [objecao]"
   Ex: "Como vender todo seu estoque em 21 dias sem liquidacao"

2. "[Numero] [coisas] que [resultado]"
   Ex: "3 frases no WhatsApp que fecham pedido no mesmo dia"

3. "O metodo de [prazo] para [resultado]"
   Ex: "O metodo de 7 dias para lotar sua agenda"
```

### Curiosidade + Dor

```
4. "Por que [acao comum] nao funciona (e o que fazer)"
   Ex: "Por que postar todo dia nao traz clientes (e o que fazer)"

5. "Voce comete esse erro de [area]?"
   Ex: "Voce comete esse erro de precificacao?"

6. "O que [autoridade] sabe sobre [assunto] que voce nao sabe"
   Ex: "O que quem fatura R$100k/mes sabe sobre vendas que voce nao sabe"
```

### Historia / Prova

```
7. "[Nome] [resultado] em [prazo]. Aqui esta como."
   Ex: "Marina faturou R$47k em 30 dias. Aqui esta como."

8. "Eu [situacao ruim]. Agora [resultado]. O que mudou:"
   Ex: "Eu nao vendia nem R$5k/mes. Agora faturo R$50k. O que mudou:"

9. "A historia de quem [resultado] comecando do [ponto baixo]"
   Ex: "A historia de quem saiu do zero a R$30k/mes vendendo no WhatsApp"
```

### Contraste / Provocacao

```
10. "Pare de [acao errada]. Comece a [acao certa]."
    Ex: "Pare de criar conteudo. Comece a criar ofertas."

11. "[Mito] e mentira. A verdade:"
    Ex: "'Precisa de muitos seguidores pra vender' e mentira. A verdade:"

12. "Voce nao precisa de [crenca popular]. Voce precisa de [verdade]."
    Ex: "Voce nao precisa de mais seguidores. Precisa de uma oferta que converta."
```

### Urgencia / Medo

```
13. "Se voce nao [acao] ate [data], [consequencia]"
    Ex: "Se voce nao mudar sua oferta ate sexta, vai perder a Black Friday"

14. "Ultima chance para [resultado]"
    Ex: "Ultima chance para entrar no metodo antes do reajuste"

15. "[Prazo] para [deadline]. O que voce vai fazer?"
    Ex: "Faltam 30 dias pro fim do ano. O que voce vai fazer?"
```

### Preemptive Claim (Hopkins)

```
16. "O unico [produto] que [claim preemptivo]"
    Ex: "O unico curso de vendas com garantia de resultado em 7 dias"

17. "[Processo exclusivo] que garante [resultado]"
    Ex: "Metodo Cascata: 4 etapas que garantem venda no primeiro contato"
```

### Mecanismo Unico (Georgi)

```
18. "Descubra o [Mecanismo Nomeado] que [resultado inesperado]"
    Ex: "Descubra o Metodo 3F que transforma DMs frias em pedidos"

19. "Por que [resultado] depende de [mecanismo], nao de [crenca]"
    Ex: "Por que vender depende do seu Script de Abertura, nao dos seus seguidores"

20. "[Mecanismo] + [Prazo] = [Resultado]"
    Ex: "Script Magn√©tico + 14 dias = agenda lotada"
```

## Checklist de Headline

- [ ] Tem beneficio claro?
- [ ] Tem numero ou prazo especifico?
- [ ] Concorrente poderia usar? (se sim, refaz)
- [ ] Trava o scroll?
- [ ] Uma unica promessa (nao multiplas)?
- [ ] Fala com UMA pessoa (nao "voces")?
- [ ] Testavel? (gere pelo menos 3 variacoes)
