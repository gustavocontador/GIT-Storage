# Task: Gerar Relatório Gerencial

**Owner:** paralegal-chief-of-staff
**Type:** Reporting
**Duration:** 2-5 minutos (background job)

## Objetivo
Gerar relatório PDF com métricas gerenciais.

## Metrics
1. Capacidade: processos ativos / capacidade total
2. Processos por status (pending, in_progress, completed)
3. Gargalos: especialistas sobrecarregados
4. Retrabalho: % processos com retrabalho
5. Clientes perdidos: capacidade insuficiente
6. Custo retrabalho: R$ mensal

## Format
- PDF multi-página
- Charts (Recharts)
- Período: últimos 30 dias
- Upload S3 + notificar user

## Output
- Relatório PDF
- Link download
