# Checklist: Alvará de Funcionamento

## PRÉ-REQUISITOS
- [ ] Empresa registrada (CNPJ ativo)
- [ ] Inscrição Municipal (ISS) ativa
- [ ] Imóvel regularizado (IPTU em dia)

---

## FASE 1: Viabilidade (2 dias)
- [ ] Zoneamento: endereço permite atividade?
- [ ] Licenças específicas por CNAE:
  - [ ] Alimentação → Alvará Sanitário
  - [ ] Saúde → Alvará Sanitário + RRT
  - [ ] Indústria → Licença Ambiental
  - [ ] Inflamáveis → AVCB
  - [ ] Eventos → Licença som
- [ ] **Checklist docs + licenças prévias**

---

## FASE 2: Documentação (3 dias)
### Empresa:
- [ ] Contrato Social + última alteração
- [ ] Cartão CNPJ
- [ ] Inscrição Municipal
- [ ] IPTU imóvel

### Imóvel:
- [ ] Contrato locação (se alugado)
- [ ] Escritura (se próprio)
- [ ] AVCB (se > 200m² ou exigido)
- [ ] Planta baixa (alguns municípios)

### Responsável Técnico (se exigido):
- [ ] RRT/ART profissional habilitado

---

## FASE 3: Protocolo (1 dia)
- [ ] Formulário prefeitura preenchido
- [ ] Docs anexados (digitalizados)
- [ ] Taxa alvará paga: R$ __________
- [ ] **Protocolo concluído (nº ________)**

---

## FASE 4: Análise (15-45 dias)
- [ ] Aguardando análise técnica
- [ ] Se pendência: complementar < 7 dias
- [ ] **Análise aprovada**

---

## FASE 5: Emissão (1 dia)
- [ ] **Alvará emitido (validade 1 ano)**
- [ ] Baixado + arquivado
- [ ] Renovação agendada (11 meses)

---

## VALIDAÇÕES
- [ ] Zoneamento OK (validado antes protocolar)
- [ ] Licenças prévias OK (sanitário, ambiental)
- [ ] Débitos municipais zerados
- [ ] Prazo: ____ dias (target 15-30)
- [ ] Custo: R$ ________ (target R$ 200-5.000)
