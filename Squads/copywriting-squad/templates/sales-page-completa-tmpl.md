## ESTRUTURA COMPLETA (12 Secoes)

### SECAO 1: PRE-HEADLINE + HEADLINE

**Pre-headline (seletor de publico):**
```
Atencao: [tipo de pessoa especifica] que [situacao/dor]
```

**Headline principal:**
```
[Promessa especifica + resultado + tempo + sem obstaculo]
```

**Subheadline:**
```
[Expandir promessa com mecanismo ou prova]
```

> **Regra Schwartz:** Headline deve ser calibrada ao nivel de awareness.
> Unaware = mais educativa. Most Aware = mais direta.

### SECAO 2: LEAD / ABERTURA (Halbert)

**Estilo carta pessoal:**
