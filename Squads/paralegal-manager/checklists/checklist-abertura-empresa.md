# Checklist: Abertura de Empresa

## PRÉ-ABERTURA
- [ ] Tipo societário definido: LTDA / EIRELI / MEI
- [ ] Nome empresarial (3 opções backup)
- [ ] Endereço sede definido
- [ ] CNAEs identificados
- [ ] Sócios definidos + % quotas
- [ ] Capital social: R$ __________

---

## FASE 1: Viabilidade (1 dia)
- [ ] Consulta nome empresarial → Aprovado
- [ ] Zoneamento endereço → Permite atividade
- [ ] CNAEs → Permitidos no município
- [ ] **Certificado Viabilidade emitido**

---

## FASE 2: Documentação (1 dia)
### Por Sócio:
- [ ] RG + CPF
- [ ] Comprovante residência (< 90 dias)
- [ ] Se casado:
  - [ ] Certidão casamento
  - [ ] RG + CPF cônjuge
  - [ ] Anuência cônjuge (regime comunhão)

### Contrato Social:
- [ ] Gerado via template
- [ ] Revisado + aprovado sócios
- [ ] Assinado (digital via Lexio ou físico)

---

## FASE 3: Registro (2 dias)
- [ ] Protocolo REI (online)
- [ ] DARE pago
- [ ] Análise Junta: aguardando
- [ ] **NIRE emitido**
- [ ] **CNPJ ativo** (automático)

---

## FASE 4: Inscrições (1 dia)
- [ ] Inscrição Municipal (ISS)
- [ ] Inscrição Estadual (ICMS) - se comércio/indústria
- [ ] GFIP (FGTS/INSS)
- [ ] Certificado digital (A1 ou A3)

---

## VALIDAÇÃO FINAL
- [ ] **Empresa 100% operacional**
- [ ] Prazo: ____ dias (target 3-5 dias)
- [ ] Custo: R$ ________ (target R$ 350-750)
