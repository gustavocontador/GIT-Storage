# Gustavo Oliveira Squad - Comandos Instalados

> Consultoria empresarial baseada no DNA de 30 anos do Gustavo Oliveira (3 Pilares: Tributário, Financeiro, Pessoas).

## Quick Start

```bash
# Diagnóstico completo de empresa
/gustavo-oliveira:agents:gustavo-diagnosticador

# Consultoria tributária
/gustavo-oliveira:agents:gustavo-tributarista

# Análise financeira
/gustavo-oliveira:agents:gustavo-financeiro
```

## Agentes Disponíveis

| Comando | Papel | Especialidade |
|---------|-------|---------------|
| `/gustavo-oliveira:agents:gustavo-chief-of-staff` | Orchestrator | Coordena o squad, triagem, roteamento |
| `/gustavo-oliveira:agents:gustavo-diagnosticador` | Diagnóstico | Avaliação completa da empresa (fiscal, financeiro, pessoas) |
| `/gustavo-oliveira:agents:gustavo-tributarista` | Tributário | Planejamento fiscal, regimes, enquadramento |
| `/gustavo-oliveira:agents:gustavo-financeiro` | Financeiro | Análise financeira, fluxo de caixa, indicadores |
| `/gustavo-oliveira:agents:gustavo-formador` | Formação | Desenvolvimento de equipes, mentoria, liderança |
