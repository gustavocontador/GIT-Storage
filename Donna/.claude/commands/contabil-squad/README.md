# Contabil Squad - Comandos Instalados

> Contabilidade completa para MEI e Simples Nacional 2026 — apuração, compliance, fiscal digital e planejamento.

## Quick Start

```bash
# Apuração de impostos (DAS, MEI)
/contabil-squad:agents:tributarista

# Verificar compliance e pendências
/contabil-squad:agents:compliance

# Relatórios gerenciais
/contabil-squad:agents:gerencial
```

## Agentes Disponíveis

| Comando | Papel | Especialidade |
|---------|-------|---------------|
| `/contabil-squad:agents:tributarista` | Tributarista | Apuração Simples/MEI, DAS, obrigações acessórias, transição IBS/CBS |
| `/contabil-squad:agents:compliance` | Compliance | Enquadramento, regularização débitos, preparação fiscalização |
| `/contabil-squad:agents:fiscal-digital` | Fiscal Digital | NF-e, NFS-e, integração ERP, leiautes SPED |
| `/contabil-squad:agents:gerencial` | Gerencial | Relatórios desempenho, projeção tributária, análise custos, comparativo regimes |
| `/contabil-squad:agents:planejamento` | Planejamento | Redução carga, créditos tributários, simulação impacto, transição 2026 |

## Localização

- **Squad completo:** `Squads/contabil-squad/`
- **Comandos slash:** `.claude/commands/contabil-squad/`
