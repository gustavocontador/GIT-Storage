# Task: Market Validation

**ID:** market-validation
**Tier:** All (but Tier 2 leads strategy, Data leads analytics)
**Duration:** 2-3 weeks
**Elicit:** false

## Objective

Get real users on MVP. Collect data. Validate that the market actually wants this. Make data-driven decisions on what to build next.

## What Happens

After MVP launches, bring in real users and measure if they love it.

- **Data Specialist:** Collect & analyze user behavior
- **Business Strategist:** Identify patterns, suggest pivots
- **Product Manager:** Collect feedback, prioritize next features
- **Developer Lead:** Fix bugs, iterate quickly
- **Mentor:** Help interpret signals

## Phase 1: User Acquisition (Days 1-7)

**Goal:** Get 100+ beta users

**Approaches:**
- [ ] Email your network
- [ ] Post on relevant communities (Reddit, Product Hunt, Hacker News)
- [ ] Write blog post explaining the product
- [ ] Talk to target customers directly
- [ ] Content marketing (Twitter, LinkedIn)
- **Target:** 100-200 signups, 30-50 daily active users

**Who does it:** You (founder) + Marketing Squad (if activated)

## Phase 2: Data Collection (Days 1-21, ongoing)

**Data Specialist tracks:**
- [ ] User signups & activation
- [ ] Feature usage patterns
- [ ] Engagement metrics
- [ ] Retention (day 1, 7, 30)
- [ ] User feedback & surveys
- [ ] Support tickets & bugs
- [ ] Revenue metrics (if charging)

## Phase 3: Analysis & Decision (Ongoing)

**Data Specialist analyzes:**
- Are users using it as expected?
- Are they finding value?
- What problems remain?
- What features matter most?
- Is revenue model working?

**Product Manager interprets:**
- What's working? Double down.
- What's not? Fix or kill.
- What's missing? Build next.

**Business Strategist checks:**
- Unit economics holding?
- Customer acquisition working?
- Retention acceptable?
- Revenue on track?

**Mentor provides wisdom:**
- "This is what the data is telling us"
- "Here's what successful founders do"
- "Should we pivot or iterate?"

## Key Metrics by Week

### Week 1 (Launch)
- 50-100 users signup
- 15-30% activation rate
- Early feedback on value

### Week 2 (Validation)
- 200+ total users
- 30%+ day-1 retention
- Clear usage patterns
- Revenue if applicable

### Week 3 (Decision Point)
- 300+ users
- 40%+ retention
- Clear signal on product-market fit
- Decision: Iterate, Pivot, or Scale

## Decision Criteria

**Continue & Iterate (Green Light):**
- [ ] 40%+ day-1 retention
- [ ] Users spending 10+ min/day
- [ ] Positive feedback on core value
- [ ] Revenue metrics on track
- [ ] Clear path to 1000+ users
- **Action:** Add more features, scale marketing

**Iterate & Investigate (Yellow Light):**
- [ ] 25-40% retention
- [ ] Mixed feedback on value
- [ ] Revenue uncertain
- [ ] Some confusion on use case
- **Action:** 10-day pivot sprint, test changes

**Pivot or Kill (Red Light):**
- [ ] <25% retention
- [ ] Negative feedback
- [ ] No paying customers
- [ ] Wrong target customer
- **Action:** Choose from opportunities #2-3, restart
