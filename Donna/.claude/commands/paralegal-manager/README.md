# Paralegal Manager Squad - Comandos Instalados

> Gestão de processos jurídicos e administrativos — licenças, holdings, certidões, alvarás, MAPA.

## Quick Start

```bash
# Coordenador geral (roteamento)
/paralegal-manager:agents:paralegal-chief-of-staff

# Triagem de novo processo
/paralegal-manager:agents:paralegal-coordinator

# Licenças MAPA (importação alimentos, registro produtos)
/paralegal-manager:agents:mapa-licencas-agent
```

## Agentes Disponíveis

| Comando | Papel | Especialidade |
|---------|-------|---------------|
| `/paralegal-manager:agents:paralegal-chief-of-staff` | Orchestrator | Coordena squad, triagem, priorização |
| `/paralegal-manager:agents:paralegal-coordinator` | Coordenação | Gestão de processos, prazos, documentos |
| `/paralegal-manager:agents:mapa-licencas-agent` | MAPA | Licenças importação, registro produtos, SIF/DIPOA |
| `/paralegal-manager:agents:holding-familiar-agent` | Holdings | Constituição holdings familiares, planejamento patrimonial/sucessório |
| `/paralegal-manager:agents:junta-comercial-agent` | Junta Comercial | Abertura/alteração/baixa empresas, registro atos societários |
| `/paralegal-manager:agents:alvara-licencas-agent` | Alvarás | Licenças funcionamento, corpo bombeiros, vigilância sanitária |
| `/paralegal-manager:agents:certidoes-compliance-agent` | Certidões | CND, FGTS, trabalhistas, compliance documental |
| `/paralegal-manager:agents:licencas-ambientais-agent` | Ambiental | Licenças ambientais (LP, LI, LO), IBAMA, outorgas |
