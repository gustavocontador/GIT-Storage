# Checklist: Licença Ambiental (LP → LI → LO)

## APLICÁVEL A:
- [ ] Indústrias (qualquer porte)
- [ ] Cervejarias
- [ ] Frigoríficos
- [ ] Atividades poluentes (efluentes, emissões)

---

## FASE 1: LP - Licença Prévia (60-90 dias)

### Documentação:
- [ ] EIA/RIMA (alto impacto) ou RCA (médio impacto)
- [ ] Memorial descritivo projeto
- [ ] Planta localização + situação
- [ ] ART engenheiro ambiental
- [ ] Certidão matrícula imóvel
- [ ] Anuência proprietário (se alugado)

### Protocolo:
- [ ] Protocolar órgão ambiental (CETESB/INEA/etc.)
- [ ] Taxa paga: R$ __________
- [ ] Análise técnica: aguardando
- [ ] Vistoria (se necessário): __/__/____
- [ ] **LP emitida (validade 2 anos)**

---

## FASE 2: LI - Licença Instalação (60-120 dias)

### Pré-Requisito:
- [ ] LP válida

### Documentação:
- [ ] Projeto executivo detalhado
- [ ] Plano gerenciamento resíduos
- [ ] Plano controle emissões atmosféricas
- [ ] Sistema tratamento efluentes (projeto)
- [ ] ART engenheiro responsável obra

### Protocolo:
- [ ] Protocolar com LP anexa
- [ ] Análise conformidade: aguardando
- [ ] **LI emitida (validade 2 anos)**

---

## FASE 3: LO - Licença Operação (30-60 dias)

### Pré-Requisito:
- [ ] Obra concluída

### Documentação:
- [ ] LI válida
- [ ] Certidão conclusão obra (Habite-se)
- [ ] Comprovação instalação sistemas controle (fotos, ART)
- [ ] Laudo operação tratamento efluentes
- [ ] Plano monitoramento ambiental
- [ ] AVCB (Corpo Bombeiros)

### Vistoria Órgão:
- [ ] Vistoria agendada: __/__/____
- [ ] Fiscal verifica instalações conforme projeto
- [ ] Resultado:
  - [ ] ✅ Aprovado → LO emitida
  - [ ] ❌ Pendências → Corrigir (30-60d)

### Emissão:
- [ ] **LO emitida (validade 2-5 anos)**
- [ ] Renovação agendada (protocolar 120 dias antes)

---

## COORDENAÇÃO CRÍTICA
- [ ] SE cervejaria MAPA: executar PARALELO (não sequencial)
- [ ] Coordenar com mapa-licencas-agent
- [ ] Economia: 3-6 meses (paralelo vs sequencial)

---

## VALIDAÇÃO FINAL
- [ ] Prazo total: ____ dias (target 150-360)
- [ ] Custo: R$ ________ (target R$ 5k-50k)
- [ ] LO válida até: __/__/____
- [ ] Renovação alerta: ativo (120 dias antes)
