## 28 GATILHOS MAPEADOS (7 Cada Tipo)

### RACIONAIS (7)

**R1: ROI Claro <30 Dias (Forca 8/10)**
- "Economiza R$ 2.5k/mes, custo R$ 12k, ROI 5 meses"
- Por que funciona: Ansiedade diminui com retorno rapido

**R2: Caso Sucesso Empresario Similar (9/10)**
- "Joao Loja X, 3 meses atras, -40% tempo, +18% faturamento"
- Elimina "nao vai funcionar comigo"

**R3: Garantia Devolucao 100% Sem Burocracia (8/10)**
- "60d teste, nao funcionar, devolvo sem perguntar"
- Elimina risco percebido

**R4: Parcelamento Sem Juros (7/10)**
- "R$ 9k → 10xR$ 900. Cabe caixa?"
- R$ 9k ativa ansiedade, R$ 900 gerenciavel

**R5: Desconto Progressivo Vista >15% (6/10)**
- So funciona caixa verde
- "R$ 8k. Vista ate sexta: R$ 6.5k (19% off)"

**R6: Comparacao Custo NAO Fazer (7/10)**
- "Trabalha 15h/sem manual = 780h/ano. 30% vendendo = R$ 35k/ano A MAIS"
- Marcos so ve custo FAZER, nao ve NAO fazer

**R7: Deadline Externa Consequencia (9/10)**
- "Cliente X ameacou sair se nao melhorar ate dia 30"
- Urgencia genuina (nao fabricada) forca

## ANATOMIA PROCRASTINACAO

### Procrastinacao Exponencial Por Valor

- R$ 1k: 1 semana
- R$ 3k: 4-6 semanas
- R$ 5k: 12 semanas
- R$ 10k: 6+ meses
- R$ 20k+: Rejeicao (nao procrastina, recusa)

### Por Que Procrastina

1. **Aversao perda extrema:** Dor gastar > prazer ganhar
2. **Ansiedade decisoria:** "E se erro?"
3. **Paralisia escolha:** Excesso opcoes trava
4. **Medo arrependimento:** Melhor nao fazer que errar
5. **Desconto hiperbolico:** Futuro abstrato vs presente concreto

## ANTI-TRIGGERS (4 Bloqueadores Automaticos)

**AT1: Vendedor Jovem Faculdade**
- Preconceito: "Nao entende PME"
- Conversao: <5%
- Bypass: Contador traz

**AT2: Linguagem Corporativa**
- "Disruptivo", "transformacao digital", "mindset"
- Rejeicao automatica
- Bypass: Linguagem direta, pratica

**AT3: Sem Caso Sucesso Verificavel**
- "Teoria bonita"
- Ceticismo maximo
- Bypass: Nome+telefone empresario validar

**AT4: Exige Mudanca Radical Rotina**
- >10h/sem esforco adicional
- Ja trabalha 70h - sem margem
- Bypass: "Economiza 20h/sem"
