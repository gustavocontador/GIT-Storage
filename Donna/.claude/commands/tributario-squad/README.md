# Tributario Squad - Comandos Instalados

> Otimização fiscal avançada — Simples Nacional, PIS/COFINS, IR/CS, incentivos regionais.

## Quick Start

```bash
# Simples Nacional (DAS, anexos, fator R)
/tributario-squad:agents:simples-nacional-specialist

# PIS/COFINS (cumulativo, não-cumulativo, créditos)
/tributario-squad:agents:pis-cofins-specialist

# IR/CS (Lucro Real, Presumido, estimativa)
/tributario-squad:agents:ir-cs-specialist
```

## Agentes Disponíveis

| Comando | Papel | Especialidade |
|---------|-------|---------------|
| `/tributario-squad:agents:simples-nacional-specialist` | Simples Nacional | DAS, anexos I-V, fator R, sublimites, exclusão/inclusão |
| `/tributario-squad:agents:pis-cofins-specialist` | PIS/COFINS | Regimes cumulativo/não-cumulativo, créditos, EFD-Contribuições |
| `/tributario-squad:agents:ir-cs-specialist` | IR/CS | Lucro Real, Presumido, estimativa mensal, JCP, incentivos |

## Localização

- **Squad completo:** `Squads/tributario-squad/`
- **Comandos slash:** `.claude/commands/tributario-squad/`
