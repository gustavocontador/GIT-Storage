# Task: Enviar Documento para Assinatura (Lexio)

**Owner:** Agent responsável
**Type:** Digital Signature
**Duration:** 5 minutos

## Objetivo
Enviar documento para assinatura digital via Lexio.

## Steps
1. Validar documento (PDF, DOC, DOCX)
2. Upload Lexio API
3. Definir signatários (nome, email, CPF)
4. Enviar notificações (email/SMS)
5. Registrar: documento pendente assinatura
6. Webhook: documento assinado → atualizar status

## Output
- Documento enviado Lexio
- Signatários notificados
- Tracking: pendente → assinado
