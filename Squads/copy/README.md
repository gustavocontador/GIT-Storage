## Fluxo de Trabalho Recomendado

```
1. /copy/copy-chief → Diagnostica o projeto (awareness, formato, complexidade)
2. copy-chief roteia para o agente ideal (ou combina multiplos)
3. Agente executa a tarefa (sales letter, VSL, landing page, etc.)
4. Revisao com checklist copy-review.md
5. Iteracao com feedback
```
