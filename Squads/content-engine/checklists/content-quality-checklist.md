# Content Quality Checklist — Master

Checklist universal de qualidade aplicavel a TODO conteudo produzido pelo Social Strategist Squad.
Cada peca de conteudo DEVE passar por este checklist antes de publicacao.

---

## Tier 1 — Positioning Alignment (BLOCKING)

Todos os itens DEVEM passar. Falha em qualquer item = conteudo bloqueado.

- [ ] Conteudo alinha com posicionamento "Clareza em meio ao caos"
- [ ] Enderea dor do avatar (profissional que ja teve resultados mas travou, obesidade mental)
- [ ] Tom e direto, sem verniz, pratico (NAO motivacional, romantico ou coach generico)
- [ ] NAO usa padroes inimigos:
  - [ ] Sem FOMO ("vagas limitadas", "so hoje", urgencia artificial)
  - [ ] Sem formulas magicas ("7 passos para o sucesso", "o segredo que ninguem conta")
  - [ ] Sem falso produtivo ("acorde as 5h", "hustle culture", "mais disciplina")
  - [ ] Sem promessas vazias ("transforme sua vida em 30 dias")
- [ ] Tem takeaway acionavel claro — se nao vira acao, nao serve
- [ ] Linguagem em PT-BR natural (nao traduzido, nao academico)
- [ ] Nao expoe dados pessoais sensiveis (endereco, financeiro, dados da filha alem do permitido)

## Tier 2 — Content Structure (BLOCKING)

Todos os itens DEVEM passar. Falha em qualquer item = conteudo bloqueado.

- [ ] Hook forte na primeira linha/slide/frase (nao comeca com contexto generico)
- [ ] Segue estrutura de framework documentado (especificar qual: _______________)
- [ ] Tem CTA claro e unico (apenas UM dos abaixo):
  - [ ] DM para conversa
  - [ ] Next Step (beta gratuito)
  - [ ] Newsletter
  - [ ] Engajamento (salvar, compartilhar, comentar)
- [ ] Respeita regra 80/20:
  - [ ] ~80% expertise (clareza, direcao, metodo, frameworks)
  - [ ] ~20% personalidade (filha, rotina, devocional, bastidores)
- [ ] Comprimento adequado para a plataforma:
  - Instagram carousel: 7-10 slides
  - Instagram caption: max 2200 chars, hook nos primeiros 125
  - YouTube script: conforme duracao-alvo
  - Newsletter: 500-1000 palavras
  - Twitter/X: max 280 chars por tweet

## Tier 3 — Quality Metrics (Scored 0-10)

Cada metrica recebe nota de 0 a 10. Minimo total: 35/50 para publicar.

| Metrica | Criterio | Nota |
|---------|----------|------|
| Hook Strength | Pattern interrupt, curiosity gap, faz parar o scroll | _/10 |
| Value Density | Insights por frase/slide, sem filler, cada frase justifica existencia | _/10 |
| Readability | Palavras simples, frases curtas, leitura fluida em voz alta | _/10 |
| Emotional Resonance | Conecta com realidade do avatar, gera identificacao | _/10 |
| CTA Effectiveness | Claro, baixa friccao, natural no contexto | _/10 |

**Total: ___/50** — Minimo para publicar: 35/50

Guia de pontuacao:
- 9-10: Excepcional, referencia para futuros conteudos
- 7-8: Bom, pronto para publicar
- 5-6: Aceitavel com ressalvas, considerar revisao
- 3-4: Abaixo do padrao, requer reescrita
- 0-2: Fora do posicionamento, descartar ou repensar

**Resultado Tier 3:** [ ] PASSA (>=35) / [ ] REPROVAR (< 35) — Total: ___/50

## Tier 4 — Platform Optimization (NON-BLOCKING)

Itens recomendados. Nao bloqueiam publicacao mas melhoram performance.

- [ ] Formato correto para a plataforma (dimensoes, duracao, tipo de arquivo)
- [ ] Hashtags/keywords adequados (se aplicavel):
  - Instagram: 5-10 hashtags, mix de volume alto e nicho
  - YouTube: titulo otimizado, tags relevantes, descricao com keywords
  - Newsletter: subject line testado
- [ ] Consistencia visual com marca:
  - Cores: B&W predominante + accent dourado (#C9A84C)
  - Fontes: Inter ExtraBold (headlines) + Courier Prime (body/typewriter)
  - Estilo: minimalista, limpo, sem poluicao visual
- [ ] Horario de publicacao adequado (picos de engajamento do avatar)
- [ ] Cross-posting planejado (repurposing para outras plataformas)

## Frases de Referencia (usar como regua de tom)

- "Voce nao precisa aprender mais. Precisa decidir melhor."
- "Clareza vem antes de disciplina."
- "O problema nao e falta de acao, e acao sem direcao."
- "Se tudo parece urgente, nada e importante."
- "Obesidade mental trava mais que preguica."

Se o conteudo nao poderia naturalmente conter uma dessas frases (ou similar), provavelmente esta fora do posicionamento.

## Auto-Correcao — Se Falhar, Faca Isso

| Falha | Correcao |
|-------|----------|
| Tier 1 — Nao alinha com posicionamento | Reescreva o hook e CTA usando uma frase-assinatura como ancora. Pergunte: "Isso resolve obesidade mental?" |
| Tier 1 — Tom motivacional/coach | Substitua adjetivos por verbos imperativos. Troque "Voce consegue!" por "Faz o seguinte:..." |
| Tier 1 — Usa padroes inimigos (FOMO, formulas) | Remova urgencia artificial. Substitua por consequencia real de nao agir |
| Tier 1 — Sem takeaway acionavel | Adicione 1 acao concreta no final: "Passo 1: [verbo] + [objeto] + [prazo]" |
| Tier 2 — Hook fraco | Reescreva usando angulo contrario ou dado surpreendente. Teste: "Isso para o scroll?" |
| Tier 2 — CTA vago ou multiplo | Escolha UM unico CTA. Remova os outros. Use formato: "[Verbo] + [onde] + [motivo]" |
| Tier 2 — Regra 80/20 violada | Conte blocos de conteudo. Se personalidade > 20%, mova para Stories |
| Tier 3 — Hook Strength < 7 | Use tecnicas: pergunta provocativa, afirmacao contraria, dado surpreendente, confissao |
| Tier 3 — Value Density < 7 | Remova frases filler. Cada frase deve ensinar, provocar ou direcionar. Se nao faz nenhuma, delete |
| Tier 3 — Readability < 7 | Leia em voz alta. Quebre frases > 20 palavras. Substitua palavras de 4+ silabas |
| Tier 3 — Emotional Resonance < 7 | Adicione cena viva do avatar. Troque descricao generica por "Voce abre o notebook e..." |
| Tier 3 — CTA Effectiveness < 7 | Conecte o CTA ao conteudo: "Se isso fez sentido, [acao]" em vez de CTA solto |
| Tier 4 — Formato incorreto | Consulte specs por plataforma na secao Tier 2 |
| Tier 4 — Hashtags inadequadas | Use mix: 2-3 alto volume + 3-4 medio + 2-3 nicho. Remova genericas (#sucesso) |
