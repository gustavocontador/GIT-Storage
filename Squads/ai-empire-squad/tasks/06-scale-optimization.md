# Task: Scale Optimization

**ID:** scale-optimization
**Tier:** All (Tier 2 strategy, Tier 3 execution)
**Duration:** Ongoing (4-week cycles)
**Elicit:** false

## Objective

Scale validated product to 1000+ users, optimize for unit economics, and position for sustainable growth toward million-dollar business.

## What Happens

Based on validation data, execute 4-week optimization sprints:

- **Product Manager:** Feature prioritization & roadmap
- **Business Strategist:** Revenue & CAC optimization
- **Data Specialist:** Growth levers & A/B testing
- **Developer Lead:** Performance & scale improvements
- **Systems Architect:** Infrastructure for 10x growth
- **DevOps Engineer:** Automation & reliability

## 4-Week Sprint Cycle

### Week 1: Analysis & Planning
**Data Specialist:**
- Analyze user cohorts & segments
- Identify retention bottlenecks
- Find biggest growth levers
- A/B test hypotheses

**Product Manager:**
- Prioritize features by impact
- Define success metrics
- Create 2-week sprint plan

**Business Strategist:**
- Calculate CAC vs LTV
- Optimize pricing if needed
- Identify expansion opportunities

**Mentor:**
- Reflect on learnings
- Adjust strategy if needed

### Weeks 2-3: Execution
**Developer Lead:**
- [ ] Highest-impact features built
- [ ] Performance optimizations deployed
- [ ] Bugs fixed from validation
- [ ] Code quality maintained

**Systems Architect & DevOps:**
- [ ] Infrastructure scales 10x
- [ ] Database optimizations
- [ ] Caching strategy
- [ ] CDN & edge optimization
- [ ] Monitoring 24/7

**Data Specialist:**
- [ ] A/B tests running
- [ ] Analytics dashboards live
- [ ] Cohort analysis ongoing
- [ ] Revenue tracking detailed

### Week 4: Review & Optimize
**Master Coordinator calls:** Sprint Review
- **Data Specialist:** Presents growth metrics
- **Product Manager:** Shipped features review
- **Business Strategist:** Revenue & CAC update
- **Systems Architect:** Performance improvements
- **Mentor:** Guidance on next sprint

**Decision:**
- [ ] Keep optimizing on same path? (Green)
- [ ] Pivot feature set? (Yellow)
- [ ] Change business model? (Red)

### Retention (Keep Users)

- Power users increasing?

### Monetization (Revenue)

- Upgrade rates acceptable?
- Enterprise opportunities?
- Expansion revenue working?

### Efficiency (Unit Economics)

- LTV growing?
- Payback period < 12 months?
- Gross margin healthy?
