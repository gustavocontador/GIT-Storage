# Task: Triagem de Processo

**Owner:** paralegal-coordinator
**Type:** Orchestration
**Duration:** 5-10 minutos

## Objetivo
Identificar tipo de processo + complexidade + rotear para especialista correto.

## Steps
1. Identificar tipo: MAPA, Holding, Junta, Alvará, Certidões, Ambiental
2. Avaliar complexidade (1-5 scale)
3. Identificar concomitância (processos paralelos)
4. Rotear para agente tier 1 especialista

## Output
- Processo classificado + roteado
- Especialista notificado
