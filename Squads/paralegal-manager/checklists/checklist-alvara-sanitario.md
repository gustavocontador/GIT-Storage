# Checklist: Alvará Sanitário (VISA)

## APLICÁVEL A:
- [ ] Restaurantes, bares, lanchonetes
- [ ] Clínicas médicas, odontológicas
- [ ] Farmácias, drogarias
- [ ] Academias, salões (alguns municípios)
- [ ] Indústrias alimentícias

---

## FASE 1: Adequação Imóvel
- [ ] Piso lavável
- [ ] Paredes lisas, impermeáveis
- [ ] Teto impermeável
- [ ] Iluminação adequada
- [ ] Ventilação adequada
- [ ] Sanitários separados (clientes vs funcionários)
- [ ] Área manipulação alimentos isolada
- [ ] Lixo com tampa acionamento pedal
- [ ] Lavatórios com sabão líquido + papel toalha

---

## FASE 2: Documentação
- [ ] Contrato Social
- [ ] Laudo água potável (se poço artesiano)
- [ ] Controle pragas (dedetização válida)
- [ ] RRT Responsável Técnico (se exigido)
- [ ] Manual Boas Práticas
- [ ] Planta baixa com fluxo processos

---

## FASE 3: Protocolo VISA
- [ ] Formulário VISA preenchido
- [ ] Docs anexados
- [ ] Taxa paga: R$ __________
- [ ] **Protocolo concluído**
- [ ] Vistoria agendada: __/__/____

---

## FASE 4: Vistoria VISA
### Checklist Pré-Vistoria:
- [ ] Imóvel 100% adequado
- [ ] Equipamentos instalados
- [ ] Higienização impecável
- [ ] Manual BPF visível
- [ ] RT presente (se aplicável)

### Vistoria:
- [ ] Fiscal VISA compareceu
- [ ] Checklist fiscal preenchido
- [ ] Resultado:
  - [ ] ✅ Aprovado → Alvará emitido
  - [ ] ❌ Reprovado → Não-conformidades listadas
