# Project Management ClickUp Squad - Comandos Instalados

> Setup avancado de ClickUp (GTD + PARA + Agile) para infoprodutos, lancamentos, conteudo, CRM e SaaS.

## Quick Start

```bash
# Orquestrador geral
/project-management-clickup:agents:pm-orchestrator

# Arquitetura de workspace
/project-management-clickup:agents:clickup-architect

# Diagnostico de processos
/project-management-clickup:agents:process-diagnostician
```

## Agentes Disponiveis

| Comando | Papel | Especialidade |
|---------|-------|---------------|
| `/project-management-clickup:agents:pm-orchestrator` | Orchestrator | Coordena todo o squad |
| `/project-management-clickup:agents:clickup-architect` | Arquiteto | Estrutura workspace, hierarquias, views |
| `/project-management-clickup:agents:process-diagnostician` | Diagnostico | Avalia processos atuais, identifica gaps |
| `/project-management-clickup:agents:automation-engineer` | Automacoes | Cria automacoes, triggers, integracoes |
| `/project-management-clickup:agents:content-operations-manager` | Conteudo | Pipeline de conteudo, calendario editorial |
| `/project-management-clickup:agents:launch-operations-manager` | Lancamentos | Checklists lancamento, timelines, dependencias |
| `/project-management-clickup:agents:crm-builder` | CRM | Pipeline vendas, lead tracking, follow-up |
| `/project-management-clickup:agents:saas-operations-specialist` | SaaS | Operacoes SaaS, sprints, backlog, roadmap |
| `/project-management-clickup:agents:support-operations-specialist` | Suporte | Tickets, SLA, base conhecimento |

## Localizacao
