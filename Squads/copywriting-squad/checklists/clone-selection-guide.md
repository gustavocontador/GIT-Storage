## FLUXO DECISORIO SIMPLIFICADO

```
PASSO 1: Qual o nivel de consciencia?
         -> Define extensao e tom

PASSO 2: Qual o tipo de produto?
         -> Define time estrategico

PASSO 3: Qual o output necessario?
         -> Define executor especifico

PASSO 4: Precisa de emocao extra?
         -> Adicionar Makepeace

PASSO 5: Precisa de bullets?
         -> Adicionar Bencivenga

PASSO 6: Precisa de email?
         -> Chaperon (auto) ou Settle (manual)

PASSO 7: Precisa de otimizacao?
         -> Hopkins + Sugarman triggers
```

## COMBINACOES A EVITAR

| COMBINACAO | POR QUE EVITAR |
|------------|----------------|
| Halbert + Carlton (mesma peca) | Estilos conflitantes (charme vs agressao) |
| Ogilvy + Makepeace | Sofisticado vs Visceral - publicos muito diferentes |
| Settle + Chaperon (mesma lista simultaneo) | Confunde frequencia e expectativa |
| Todos os Sugarman triggers de uma vez | Exagero diminui impacto |

## REGRA DE OURO

> **Nunca comece a escrever copy sem passar por pelo menos 1 estrategista.**

> O erro mais comum e pular direto para execucao sem ter clareza sobre awareness do mercado, mecanismo unico, ou estrutura da oferta.
