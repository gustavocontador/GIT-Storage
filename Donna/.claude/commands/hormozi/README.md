# Hormozi Squad - Comandos Instalados

> 16 especialistas baseados em Alex Hormozi: $100M Offers, Leads, pricing, funnels, escala, retencao.

## Quick Start

```bash
# Orchestrator — roteia pro especialista certo
/hormozi:agents:hormozi-chief

# Criar oferta irresistivel ($100M Offers)
/hormozi:agents:hormozi-offers

# Gerar leads ($100M Leads)
/hormozi:agents:hormozi-leads

# Precificacao e modelos de receita
/hormozi:agents:hormozi-pricing
```
