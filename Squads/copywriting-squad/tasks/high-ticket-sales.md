## Overview

Workflow completo para criar copy de vendas high-ticket (acima de R$5.000). Combina estrategia profunda de avatar (Kennedy), construcao de oferta irresistivel (Hormozi), mecanismo unico (Todd Brown), e execucao de sales letter longa (Halbert/Makepeace). Inclui sequencia de aplicacao e script de call de vendas.

## Quando Usar

- Mentoria ou consultoria premium (R$5k-50k+)
- Programa de coaching high-ticket
- Servico B2B de alto valor
- Masterminds e experiencias exclusivas
- Qualquer oferta que vende via call/aplicacao

---

## Input

- **Oferta** (obrigatorio): Descricao detalhada do produto/servico high-ticket
- **Preco** (obrigatorio): Valor ou faixa de preco
- **Avatar** (obrigatorio): Quem e o comprador ideal (quanto mais detalhado, melhor)
- **Mecanismo de Venda** (obrigatorio): Call, aplicacao, webinar, VSL → call
- **Resultados/Cases** (recomendado): Provas de resultado de clientes anteriores
- **Objecoes Conhecidas** (recomendado): Principais razoes de nao comprar

## Output

- **Sales Letter Longa**: 3000-5000 palavras, formatada para web
- **VSL Script**: 20-45 min, estruturado para video
- **Sequencia de Aplicacao**: 3-5 emails pre-call
- **Script de Call**: Roteiro de venda consultiva
- **Objection Handling Doc**: Respostas para top 10 objecoes

## Action Items

### Step 1: Avatar Profundo (4Ms)
**Clone:** @dan-kennedy

1. Aplicar framework 4Ms (Market, Message, Media, Moment)
2. Responder 20 perguntas profundas sobre o avatar
3. Mapear dores emocionais (nao so racionais)
4. Identificar objecoes pre-existentes
5. Definir linguagem exata do mercado

**Output:** Avatar completo com linguagem mapeada

---

### Step 2: Oferta Grand Slam
**Clone:** @alex-hormozi

1. Aplicar Value Equation: Resultado Desejado × Probabilidade / Tempo × Esforco
2. Criar stack de valor com precificacao individual
3. Definir bonus estrategicos (que reduzem objecoes)
4. Criar garantia que elimina risco
5. Ancorar preco vs valor total do stack

**Output:** Oferta completa com stack de valor

---

### Step 3: Unique Mechanism
**Clone:** @todd-brown

1. Identificar por que solucoes anteriores falharam para o avatar
2. Criar mecanismo do problema (por que esta preso)
3. Criar mecanismo da solucao (por que ISSO funciona)
4. Nomear o metodo/sistema (proprietary name)
5. Criar Big Idea que conecta mecanismo + resultado

**Output:** Mecanismo unico + Big Idea nomeada

---

### Step 4: RMBC Brief
**Clone:** @stefan-georgi

1. Compilar Research (linguagem do mercado, dores, desejos)
2. Documentar Mechanism (do Step 3)
3. Criar Brief completo (respostas em formato copy)
4. Definir estrutura e secoes do copy

**Output:** RMBC document completo

---

### Step 5: Sales Letter Longa
**Clone:** @gary-halbert (narrativa) + @clayton-makepeace (emocao)

1. Headline principal (baseada em awareness level)
2. Lead: historia de identificacao (Halbert)
3. Problema amplificado com emocao visceral (Makepeace)
4. Transicao para solucao via mecanismo unico
5. Apresentacao da oferta com stack de valor
6. Proof section (cases, depoimentos, credenciais)
7. Close com urgencia e CTA para aplicacao/call
8. P.S. estrategicos (minimo 2)

**Output:** Sales letter 3000-5000 palavras

---

### Step 6: VSL Script
**Clone:** @jon-benson

1. Aplicar 5-Step Formula adaptada para high-ticket
2. Hook de abertura (primeiros 30 segundos criticos)
3. Story-driven middle (conexao emocional)
4. Mechanism reveal (momento de "aha")
5. Oferta + stack (visual para slides)
6. CTA para aplicacao (nao venda direta)

**Output:** Script de VSL 20-45 min com marcacoes de slide

---

### Step 7: Sequencia de Aplicacao + Script de Call
**Clone:** @andre-chaperon (emails) + @dan-kennedy (call)

**Emails (Chaperon):**
1. Email 1: Confirmacao + expectativa
2. Email 2: Case study relevante
3. Email 3: Pre-frame para a call (o que esperar)
4. Email 4: Reminder + urgencia (se nao apareceu)

**Script de Call (Kennedy):**
1. Abertura consultiva (nao vender, diagnosticar)
2. Perguntas de qualificacao (dor + desejo + capacidade)
3. Apresentacao da solucao (if qualified)
4. Objection handling (top 5)
5. Close (proximo passo claro)

**Output:** 4 emails + script de call

## Acceptance Criteria

- [ ] **AC1:** Avatar documentado com 4Ms aplicado e linguagem mapeada
- [ ] **AC2:** Oferta com Value Equation e stack de valor completo
- [ ] **AC3:** Mecanismo unico nomeado com Big Idea clara
- [ ] **AC4:** RMBC document completo
- [ ] **AC5:** Sales letter 3000+ palavras com todas as secoes
- [ ] **AC6:** VSL script 20+ min com marcacoes de slide
- [ ] **AC7:** Sequencia de 3-5 emails de aplicacao
- [ ] **AC8:** Script de call com objection handling

## Veto Conditions

- **VETO-HTS-1:** Copy sem avatar profundo (4Ms) → PARAR, voltar ao Step 1
- **VETO-HTS-2:** Oferta sem Value Equation → PARAR, voltar ao Step 2
- **VETO-HTS-3:** Sales letter sem mecanismo unico → PARAR, voltar ao Step 3
- **VETO-HTS-4:** CTA direciona para compra direta (high-ticket vende via call) → CORRIGIR para aplicacao
- **VETO-HTS-5:** Copy sem proof section (cases/depoimentos) → ADICIONAR antes de entregar

## Anti-Patterns

- Vender high-ticket com copy curto de e-commerce — precisa ser longo e profundo
- Pular avatar profundo e usar persona generica — high-ticket exige intimidade com o avatar
- Usar urgencia falsa (timers fakes) — publico high-ticket detecta e desconfia
- Fazer CTA direto para pagamento — high-ticket vende via call/aplicacao
- Ignorar objection handling — cada objecao nao respondida e uma venda perdida
- Copy que fala sobre features em vez de transformacao — high-ticket vende resultado
