# Content Distillery Squad - Comandos Instalados

> Transforma videos/lives do YouTube em frameworks, resumos e 60+ pecas de conteudo.

## Quick Start

```bash
# Orchestrator — pipeline completo
/content-distillery:agents:distillery-chief

# Extrair conhecimento tacito
/content-distillery:agents:tacit-extractor

# Identificar frameworks e modelos mentais
/content-distillery:agents:model-identifier
```
