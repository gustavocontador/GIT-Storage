# Task: Acompanhamento de Processo

**Owner:** Agent especialista
**Type:** Monitoring
**Duration:** Contínuo (até conclusão)

## Objetivo
Acompanhar processo após protocolo até conclusão.

## Steps
1. Monitorar status online (portais órgãos)
2. Se pendência: resolver < 7 dias
3. Atualizar status sistema (real-time dashboard)
4. Notificar cliente mudanças (Onvio)
5. Se aprovado: baixar documentos finais

## Output
- Status atualizado
- Cliente informado
- Docs finais arquivados
